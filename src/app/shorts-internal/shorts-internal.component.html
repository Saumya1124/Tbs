Carousel Upgrades

Improvements made to the default Bootstrap 4 Carousel, which provide greater accessibililty through keyboard navigation, lazy loading and normalized slide heights across a variety of image sizes.

  • Tab to Carousel to navigate slides using keyboard
  • Slides can be changed using arrow keys, number keys, or by tabbing to slide controls and pressing enter
  • Red outline ONLY used to show current selection
